Projects

  • gameboy-emu

    GameBoy emulator written in C++. Emulating Ziglog Z80 (really is a Sharp LR35902 with some new features and some removed ones) cycle accurately. (under construction :D)

  • voxelcraft

    OpenGL minecraft clone written in C++. Infinite worlds, varrying terrain through noise, block placement and destruction, ray casting, different block types, and other cool OpenGL stuff. Update Videos

  • isometric terrain generation

    Isometric terrain generation written in C++ and SFML. Heavly inspired my minecraft's chunking system and usage of noise for procedural generation. Currently supports infinite worlds and control over terrain generation. Demo

  • asteroids

    • Clone of classic Atari game, Asteroids. Written in C++ using the SFML Library. Implemented Lorentz Factor to accurately represent space movement. Implemeneted SAT Collision. Demo

  • hashmap-c

    Hashmap simulation created in C. Allows for all basic hashmap operations. Created to be used with strings. Boasts dynamic resizing, djb2 hashing function, accessable insertion and deletion.
