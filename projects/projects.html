Projects

  • bsqgc

    Ongoing research for Bosque's generational garbage collector written in modern C. Designed to boast a reference counted old space and compacted young space.

  • gameboy

    Gameboy emulator written in C. I initially wrote this in C++ and made some of the most horrific design choices ever. The CPU for my C port went from ~2300 LOC to ~1200 LOC. This emulator isn't perfect by any means, and I have a version I am developing currently designed for much closer hardware accuracy and a very nice debugging system.

  • chip8

    Chip 8 emulator written in C89 (yikes).

  • voxelcraft

    OpenGL minecraft clone written in C++. Infinite worlds, varrying terrain through noise, block placement and destruction, ray casting, different block types, and other cool OpenGL stuff. Update Videos

  • isometric terrain generation

    Isometric terrain generation written in C++ and SFML. Heavly inspired my minecraft's chunking system and usage of noise for procedural generation. Currently supports infinite worlds and control over terrain generation. Demo
